## 一、项目概述

**项目名称：** AI API 中转平台（品牌名待定）

**项目类型：** 全栈完整平台（落地页 + 用户系统 + 控制台 + 后端）

**核心定位：** 一站式 AI 模型接入平台，统一管理多家大模型供应商，兼容 OpenAI 格式，为个人与企业提供集中式模型管理与网关服务。

**目标用户：** 个人开发者、学生、技术爱好者，以及中小企业、创业团队。

---

## 二、设计方向

### 视觉风格：科技感 / 深色主题

### 配色方案：蓝紫渐变

- **主色调：** 深色背景 (#0a0f1a / #111827)
- **渐变色：** 蓝色 (#3b82f6) → 紫色 (#8b5cf6) → 粉紫 (#a855f7)
- **强调色：** 亮蓝色光效 (#60a5fa)
- **文字色：** 白色主文字 (#ffffff)、灰色次要文字 (#94a3b8)
- **边框/分割：** 深灰 (#1f2937)

### 动效风格：高动态

- 粒子效果背景
- 背景动态光效（渐变流动）
- 数字滚动动画
- 悬浮交互反馈
- 滚动触发的入场动画

## 三、页面结构

### 1. 导航栏（透明毛玻璃型）

- Logo + 品牌名（左侧）
- 主导航：首页、控制台、模型广场、文档、关于
- 右侧：通知图标（显示数量）+ 用户头像 + 账户下拉菜单
- 滚动时背景变为磨砂玻璃效果（backdrop-blur）
- 高度：64px，固定定位

### 2. Hero 首屏区域（左右分栏）

**左侧（约 55%）：**
- 主标题：大号白色字体，分两行
- 副标题：蓝色渐变字体，描述核心价值
- 行动按钮：主按钮（立即开始，蓝色渐变）+ 次按钮（查看文档，白色描边）
- API 接口地址展示框（可一键复制）
- 数据统计：服务可用率 99.9% / 支持模型 500+ / 平均响应 <99ms / 7×24 技术支持

**右侧（约 45%）：**
- 动态背景效果（粒子/光效）
- 或抽象的 AI/网络相关图形

### 3. 为什么选择我们（8 个优势卡片）

每张卡片包含：图标 + 标题 + 简短描述

1. 🚀 极速响应 — 自研智能路由，国内节点优化
2. 🛡️ 稳定可靠 — 99.9% 可用率保障
3. 💰 价格实惠 — 比官方更低还有赠送
4. 🔗 兼容性强 — 完全兼容 OpenAI API
5. 🌐 模型丰富 — 500+ 主流模型
6. 👨‍💻 技术支持 — 7×24 小时在线
7. 📊 用量清晰 — 实时统计明细
8. 🔒 安全合规 — 隐私保护

**布局：** 2 行 × 4 列网格，卡片深色背景 + 边框光效

### 4. 支持的模型厂商

以 Logo 图标墙形式展示，支持的品牌：

OpenAI、Anthropic、DeepSeek、Google、Meta、Mistral、Moonshot、xAI、字节跳动、智谱、阿里巴巴 等

**布局：** 水平排列，多行自适应，颜色可灰度处理，hover 时彩色

### 5. 用户评价（好评展示）

8 张用户评价卡片，姓氏 + 身份 + 五星 + 评价内容

### 6. 代理商招募（可选模块）

- 标题：加入我们，成为专属代理商
- 四大优势：高额折扣 / 专属后台 / 适合人群 / 技术支持
- 代理商要求（6 条）
- 入驻流程（4 步：提交申请 → 资质审核 → 搭建网站 → 正式入驻）
- 行动按钮：点击立即申请

### 7. Footer

- 版权信息
- 系统由 New API 驱动
- 联系方式（QQ / 微信）
- 底部链接

## 四、技术实现

### 前端

- **框架：** 纯 HTML + CSS + JavaScript（单页落地页）
- **动画：** CSS 动画 + Canvas 粒子效果
- **图标：** Lucide Icons / Emoji
- **字体：** 系统字体栈（无外部字体依赖）

### 后端（后续阶段）

- 用户注册 / 登录系统
- 余额与充值系统
- API Key 管理
- 模型中转网关（基于 New API 或自建）
